How to use:

  1. Load Primary Dataset: Upload your main JSON file containing benchmark results or paste the data directly
  2. Optional Comparison: Upload a second JSON file to compare results side-by-side (different colors and shapes)
  3. Add Version Labels: Customize the dataset labels to identify versions (e.g., "v1.0", "Before optimization")
  4. Select Metrics: Choose the performance metric for Y-axis and what to display on X-axis
  5. Group Data: Group results by engine, dataset, or setup to compare different configurations
  6. Filter Results: Use filters to focus on specific engines, datasets, or setups
  7. Switch Views: Change between line charts, scatter plots, and bar charts
  8. Track Sources: Hover over data points to see filename, version, engine, and HNSW configuration details
  9. Compare Files: In comparison mode, primary data shows as blue circles, comparison data as orange triangles
